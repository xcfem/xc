# Sections dimensions and properties from different suppliers and standards

This folder contains the JSON files where the desing properties of different steel profiles are stored. Most of those properties have been obtained from suppliers handbooks like those in the references. Anyway, the preferred way of generating the design properties is through formulae (see en_10210-2_en_10219-2/ folder in this directory).

## References

- [SCI Blue Book](https://www.steelforlifebluebook.co.uk/)
- [Arcelor Orange Book](https://orangebook.arcelormittal.com/)
- [Eurocode applied EC3](https://eurocodeapplied.com/design/en1993)
